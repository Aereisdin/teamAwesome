HTML

As you start out in your coding adventure, you're going to start with HTML more than likely. HTML stands for HyperText Markup Language and it is the structure of web browsers. It can be assisted by technologies such as Cascading Style Sheets (CSS) and script languages such as JavaScript. Below are examples of the basic tags that are used in HTML pages.

Heading Tags

Any document starts with a heading. You can use different sizes for your headings. HTML also has six levels of headings, which use the elements h and 1-6. The smaller the number, the larger the text will be. While displaying any heading, browser adds one line before and one line after that heading.

Paragraph Tag

The p tag offers a way to structure your text into different paragraphs. Each paragraph of text should go in between an opening "p" and a closing "/p" tag
